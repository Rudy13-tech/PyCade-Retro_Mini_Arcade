# Retro-Mini-Arcade
Follow this steps to use it:-
1. Download the ZIP file of this repo.
2. Extract all the things into one folder.
3. Open and run "Main.py".
4. A window will open and display and URL.
5. Open that URL in your Mobile.(Make sure both devices are connected to one network)
6. Start playing with your mobile/tablet as controller..!.
